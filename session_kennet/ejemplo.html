Ejemplo: iliberalismo en discursos políticos

Código tomado del ejemplo oficial de quallmer (Maerz & Schneider 2020)

Published

April 23, 2026

Basado en la viñeta Example: Illiberalism in political speeches. El artículo original usa 4.740 discursos desde un .rds; aquí usamos la muestra incluida en quallmer (data_corpus_ms2020sample, 100 discursos) y el índice de diccionario ya viene en score.

Maerz, S. F., & Schneider, C. Q. (2020). Quality & Quantity, 54, 517–545. https://doi.org/10.1007/s11135-019-00885-7

Codebook (liberal–iliberal)

codebook_ideology <- qlm_codebook(
  name = "Liberal-illiberal rhetoric",
  instructions = paste(
    "Analyze the rhetorical style of this political speech.",
    "",
    "ILLIBERAL rhetoric (negative scores) includes:",
    "- Nationalism and patriotic appeals",
    "- Paternalism and appeals to tradition",
    "- Emphasis on order, stability, and security",
    "- In-group/out-group distinctions",
    "- Rejection of pluralism",
    "",
    "LIBERAL rhetoric (positive scores) includes:",
    "- Individual rights and freedoms",
    "- Tolerance and pluralism",
    "- Civil liberties and minority rights",
    "- Democratic values and rule of law",
    "- Open society principles",
    "",
    "A score of 0 indicates neutral or mixed rhetoric."
  ),
  schema = ellmer::type_object(
    score = ellmer::type_integer(
      description = "Rhetoric score from -10 (illiberal) to +10 (liberal)"
    ),
    explanation = ellmer::type_string(
      description = "Brief explanation of the assigned score"
    )
  ),
  role = "You are an expert political scientist analyzing political rhetoric.",
  input_type = "text"
)

Codificación con el LLM

speech_texts <- work$text
names(speech_texts) <- work$.id

coded_speeches <- qlm_code(
  speech_texts,
  codebook = codebook_ideology,
  model = "openai/gpt-4o-mini",
  name = "gpt4o_mini_ideology",
  notes = sprintf(
    "LLM coding of %d speeches (subset of data_corpus_ms2020sample) for liberal-illiberal rhetoric",
    length(speech_texts)
  )
)

Agregación a nivel de orador

coded_with_meta <- work |>
  dplyr::select(.id, speaker, country, regime, dictionary_score = score) |>
  dplyr::left_join(
    as.data.frame(coded_speeches) |> dplyr::select(.id, llm_score = score),
    by = ".id"
  )

speaker_scores <- coded_with_meta |>
  dplyr::group_by(speaker, country, regime) |>
  dplyr::summarise(
    n_speeches = dplyr::n(),
    dictionary_score = mean(dictionary_score),
    llm_score = mean(llm_score, na.rm = TRUE),
    .groups = "drop"
  ) |>
  dplyr::mutate(
    dictionary_z = as.numeric(scale(dictionary_score)[, 1]),
    llm_z = as.numeric(scale(llm_score)[, 1])
  ) |>
  dplyr::arrange(dictionary_score)

Comparación diccionario vs. LLM (qlm_compare)

dictionary_coded <- as_qlm_coded(
  speaker_scores |> dplyr::select(.id = speaker, score = dictionary_z),
  name = "dictionary_ms2020"
)

llm_coded <- as_qlm_coded(
  speaker_scores |> dplyr::select(.id = speaker, score = llm_z),
  name = "gpt4o_mini_aggregated"
)

comparison <- qlm_compare(
  dictionary_coded,
  llm_coded,
  by = "score",
  level = "interval",
  tolerance = 1
)

Figura tipo artículo (z-scores)

plot_data <- speaker_scores |>
  tidyr::pivot_longer(
    cols = c(dictionary_z, llm_z),
    names_to = "method",
    values_to = "score"
  ) |>
  dplyr::mutate(
    method = dplyr::case_when(
      method == "dictionary_z" ~ "Dictionary (original)",
      method == "llm_z" ~ "LLM (GPT-4o-mini)"
    )
  )

ggplot(plot_data, aes(x = reorder(speaker, score), y = score, color = regime, shape = method)) +
  geom_line(aes(group = speaker), linewidth = 0.3, alpha = 0.5) +
  geom_point(size = 3, alpha = 0.8) +
  geom_hline(yintercept = 0, linetype = "dashed", alpha = 0.5) +
  scale_color_manual(
    name = "Regime type",
    values = c("Autocracy" = "#FF3300", "Democracy" = "#00CC00")
  ) +
  scale_shape_manual(
    name = "Method",
    values = c("Dictionary (original)" = 16, "LLM (GPT-4o-mini)" = 17)
  ) +
  labs(
    x = NULL,
    y = "Illiberal \u2194 Liberal rhetoric (z-score)",
    title = "Ideology scores by speaker: Dictionary vs. LLM",
    subtitle = "Both scores standardized to z-scores for comparison"
  ) +
  coord_flip() +
  theme_minimal() +
  theme(
    legend.position = "bottom",
    legend.box = "vertical",
    panel.grid.major.y = element_blank()
  )

Conclusión (como en la viñeta)

  1. Resultados comparables entre diccionario y LLM (correlación / acuerdo).
  2. El LLM capta matices retóricos que el diccionario no cuenta palabra a palabra.
  3. Las explicaciones del modelo ayudan a interpretar y auditar.
  4. El diccionario sigue siendo transparente; el LLM puede documentarse con qlm_trail().
